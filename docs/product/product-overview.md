# Product Overview

## 项目定位

OpenClaw 多 Agent 平台面向个人和团队用户，提供 Agent 上传、克隆、对话、组合、团队协作和市场化分发能力。平台将每个 Agent 视为一个可运行、可复用、可演进的 workspace 资产，并围绕 workspace 建立市场、项目、团队和运行记录。

## 核心用户

- 普通用户：浏览市场、领取或上传 Agent、创建项目、与单个 Agent 对话、组织团队完成任务。
- 创作者：发布 Agent、团队模板、场景化工作流和方法论，并通过市场分发获得持续回报。
- 管理者：审核市场内容、管理供给质量、观察运行数据和处理异常资源。

## 核心能力

- Agent Market：展示官方和用户发布的 Agent、模板和场景能力。
- My Agents：管理用户私有 Agent workspace，与单 Agent 进行独立对话。
- Team Workflow：通过自然语言或画布创建多 Agent 团队，并执行团队任务。
- Project Workspace：把任务、文件、Agent、团队运行和交付物收敛到项目空间。
- Tea Party：提供轻量的多 Agent 群聊协作场景。
- Desktop Mode：在本机扫描和导入 Claude Code、Codex、OpenCode、Hermes、OpenClaw 等本地 Agent workspace。

## 下一阶段市场化方向

平台后续将从资源列表推进为能力供给、模型资源与服务流转平台。重点包括创作者主页、版本管理、能力标签、模板包装、Token 中转、官方模型 Key、订阅额度包、组织采购和创作者分成。

市场化目标不是简单增加 Agent 数量，而是让优质 Agent、团队模板和场景工作流能够被稳定包装、持续更新、高效分发，并让上传优质能力的创作者获得正向回报。
