# Demo Script

## 演示目标

用一条完整路径展示平台如何从发现 Agent、创建项目、组织团队，到执行多 Agent 协作任务并查看结果。

## 演示流程

1. 进入首页，展示宽屏传统模式和专业模式切换。
2. 打开 Agent Market，浏览官方 Agent 或用户发布 Agent。
3. 领取或上传一个 Agent，进入 Agent 设置流程。
4. 在 My Agents 中打开单 Agent 对话，发送测试消息并确认回复。
5. 进入 Projects，创建项目 workspace，并绑定相关 Agent 或团队。
6. 进入 Architectures，使用自然语言或画布创建团队。
7. 运行团队任务，展示节点状态、运行事件、handoff 文件和 artifacts。
8. 进入 Tea Party，展示多 Agent 群聊协作。
9. 展示桌面端导入本地 Agent 的入口和扫描结果。
10. 展示部署后的公网访问、登录、WebSocket 和后端健康检查。

## 演示强调点

- 单 Agent 对话不会污染团队运行 workspace。
- 团队运行结果通过 runtime workspace、artifacts 和 handoff 文件传递。
- 市场资源、用户私有资产、项目空间和团队运行空间有明确边界。
- AI 协作规范已经沉淀在 `docs/ai-collaboration/`，后续开发优先遵循 spec。
